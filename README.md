# 🐾 Pet Adoption System

## 📌 Project Description
The Pet Adoption System is a Database Management System (DBMS) project developed using MySQL. It helps manage pet records, customer details, employee information, adoptions, and payment records.

## ✨ Features
- Pet Management
- Customer Management
- Employee Management
- Adoption Records
- Payment Management
- SQL Queries
- Views
- Triggers

## 🛠 Technologies Used
- MySQL
- SQL
- Visual Studio Code
- Git
- GitHub
